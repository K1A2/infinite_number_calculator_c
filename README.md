# 무한소수 계산기

## 역할

* 코딩파트 - 김준철, 김은채, 김민성
* 파일파트 - 이창민, 최선아

## 기능

### 입력

1. input 파일로 부터 계산 식 읽어오기
   * 이때 문자열의 길이 제한 x -> 동적할당을 위해 **연결 리스트**로 구현
2. 입력에 대한 예외처리 필수
3. 곱하기 기호 생략의 예외 처리 (ex. 2(3+2) = 2*(3+2))

### 계산

1. 수식 표기법을 중위표기법에서 후위표기법으로 변환
   * **스택**을 이용
2. 덧셈/곱셈/뺄셈 함수 구현. 나눗셈은 필수는 아님
   * **연산의 정확도**가 평가 기준

## 평가 기준

연산의 정확도. 3가지 항목으로 평가

1. 결과의 정확도 (오차)
2. 괄호 처리 여부, 입력의 다양성, 예외처리 등
3. 나눗셈 구현 시 추가 점수
